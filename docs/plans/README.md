# Kế hoạch triển khai (`docs/plans/`)

Thư mục này dùng lưu **file plan** (Markdown) khi áp dụng skill **ke-hoach-trien-khai** / **thuc-hien-ke-hoach** (xem `.agents/skills/`).

## Quy ước đặt tên

`YYYY-MM-DD-<ten-tinh-nang>.md`

Ví dụ: `2026-03-28-them-filter-bao-cao.md`

File `.gitkeep` giữ thư mục trong Git khi chưa có plan nào.
